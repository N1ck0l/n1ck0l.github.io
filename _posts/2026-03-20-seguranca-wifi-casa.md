Sua rede WiFi é tão importante quanto as portas e janelas da sua casa. Se não estiver protegida, invasores podem entrar. Aqui está um guia completo para securizá-la.

## 🔒 O Perigo Real

Uma rede WiFi desprotegida permite que qualquer pessoa:
- 🚨 Roube sua senha de banco
- 📷 Espie seu tráfego
- 💻 Use sua internet para atividades ilícitas
- 📡 Acesse seus arquivos compartilhados

## ✅ 5 Passos Essenciais

### 1️⃣ Mude a Senha do Roteador (Admin)

```bash
URL: 192.168.1.1 ou 192.168.0.1
Usuário: admin
Senha PADRÃO: admin ou 12345
# MUDE IMEDIATAMENTE!
```

**Por que?** 99% dos roteadores vêm com senha padrão. Qualquer um pode acessar.

### 2️⃣ Ative Criptografia WPA3 (ou WPA2)

Na configuração do roteador:
- ❌ EVITE: WEP ou WPA (obsoletos)
- ✅ USE: WPA3 (mais seguro) ou WPA2 (compatível)

```
Exemplo de senha forte:
M!7k$9xQp@2nRvL#5tWqK&8jHfYuBn3I4m
```

### 3️⃣ Crie uma Senha WiFi Forte

- Mínimo 16 caracteres
- Combine maiúsculas, minúsculas, números e símbolos
- Evite datas pessoais, nomes ou sequências

**Teste sua senha:** [https://www.kaspersky.com/password-checker](https://www.kaspersky.com/password-checker)

### 4️⃣ Desabilite WPS (WiFi Protected Setup)

WPS tem vulnerabilidades conhecidas. Vá em:
```
Configurações → Segurança → Modo WPS → Desabilitar
```

### 5️⃣ Atualize o Firmware do Roteador

- Verifique a marca do seu roteador
- Procure por "atualização de firmware" no manual
- Instale a versão mais recente

## 🛡️ Bônus: Configurações Avançadas

### Oculte a Rede (SSID Broadcast)

- Desabilite broadcast do SSID
- Sua rede não aparece na lista pública
- Segurança por obscuridade (fraca, mas válida)

### Ative Firewall do Roteador

```
Configurações → Firewall → Ativar
```

### Configure Salão de Hóspedes

Para visitantes, crie rede separada:
- Sem acesso aos seus compartilhamentos
- Senha diferente e alterável

## 📊 Checklist Final

| Item | Status |
|------|--------|
| Senha do admin roteador mudada | ☐ |
| WiFi com WPA3/WPA2 | ☐ |
| Senha WiFi forte (16+ chars) | ☐ |
| WPS desabilidado | ☐ |
| Firmware atualizado | ☐ |
| Firewall ativo | ☐ |
| Rede de hóspedes configurada | ☐ |

## 💡 Dica Pro

Configure seu roteador com IP estático em seu notebook e faça backup das configurações. Se algo der errado, você terá um restore point.
